# bme-knowledge-base
Physics simulates realistic movement and interactions between different objects. Without a proper physics engine game would not be playable and charachter controls wouldn't feel comfortable.
Languages: C++, C# 
Technologies: PhysX, Box2D, Bullet, Unity Physics. 
Core concepts: collision detection, gravity, forces, momentum, rigid body dynamics.
Just like other concepts coding a physics engine from scratch is a hard work so generally game engines conatins physic engines in it.
