# bme-knowledge-base
Graphics programming deals with rendering rendering, shaders, lighting, sprites, and performance optimization. 
One of the most important aspects in game devolopment due to our eyes being primary sensory organ.
Common languages: C++, C#, GLSL, HLSL.
Popular technologies: DirectX, Vulkan, OpenGL, Web GL.
Modern Game and Graphic engines manually render those for programmer but changing style, lightning or adding unique styles still up to devoloper.
